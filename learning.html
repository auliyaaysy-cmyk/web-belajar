1. HTML (Struktur / Kerangka) HTML (HyperText Markup Language) adalah pondasi sebuah web. Gunanya untuk menentukan apa isi dari web tersebut. Analogi: Kerangka rumah (tembok, pintu, jendela). Tag Penting:

Tanpa CSS, website hanya akan berisi teks hitam-putih yang membosankan. Analogi: Cat tembok, dekorasi, posisi furnitur, dan lampu. Fungsi Utama: Mengatur warna teks dan background. Mengatur ukuran font dan jarak (margin/padding). Membuat tata letak agar rapi (tengah, kiri, atau kanan). 3. JavaScript (Logika / Interaksi) JavaScript adalah bahasa pemrograman yang membuat website "hidup" dan bisa berpikir. Analogi: Sistem kelistrikan atau sensor otomatis (pintu terbuka sendiri, lampu menyala saat gelap). Fungsi Utama: Membuat tombol yang bisa memunculkan pesan. Membuat slider gambar yang bergeser otomatis. Menghitung angka atau memproses formulir pendaftaran.
